# Delete Comment
Source: [https://docs.gitcode.com/en/docs/apis/delete-api-v-5-repos-owner-repo-pulls-comments-id](https://docs.gitcode.com/en/docs/apis/delete-api-v-5-repos-owner-repo-pulls-comments-id)
Category: Pull Requests
- Method: `DELETE`
- Path: `/api/v5/repos/{owner}/{repo}/pulls/comments/{id}`
- Operation ID: `delete_api_v5_repos_{owner}_{repo}_pulls_comments_{id}`
- Tags: `Pull Requests`
- Deprecated: `false`
## Parameters
| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| owner | path | true | string | Repository space address (organization or individual's address path) |
| repo | path | true | string | Repository path (path) |
| id | path | true | integer | The ID of the comment. |
| access_token | query | true | string | User authorization code |
## Request Body
No request body.

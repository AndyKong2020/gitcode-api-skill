# Get the diff of commit
Source: [https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-commits-sha-diff](https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-commits-sha-diff)
Category: Commit
- Method: `GET`
- Path: `/api/v5/repos/{owner}/{repo}/commit/{sha}/diff`
- Operation ID: `get_api_v5_repos_{owner}_{repo}_commits_{sha}_diff`
- Tags: `Commit`
- Deprecated: `false`
## Parameters
| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| owner | path | true | string | Repository space address (organization or individual's address path) |
| repo | path | true | string | Repository path (path) |
| sha | path | true | string | commit's SHA |
| access_token | query | true | string | User authorization code |
## Request Body
No request body.

Successful Response

# Get Repository Languages
Source: [https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-languages](https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-languages)
Category: Repositories
- Method: `GET`
- Path: `/api/v5/repos/{owner}/{repo}/languages`
- Operation ID: `get_api_v5_repos_{owner}_{repo}_languages`
- Tags: `Repositories`
- Deprecated: `false`
## Parameters
| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| owner | path | true | string | Repository space address (organization or individual's address path) |
| repo | path | true | string | Repository path (path) |
| access_token | query | true | string | User authorization code |
## Request Body
No request body.

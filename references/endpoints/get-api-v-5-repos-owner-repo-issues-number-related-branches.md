# Get the list of branches associated with the issue
Source: [https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-issues-number-related-branches](https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-issues-number-related-branches)
Category: Issues
- Method: `GET`
- Path: `/api/v5/repos/{owner}/{repo}/issues/{number}/related_branches`
- Operation ID: `get_api_v5_repos_{owner}_{repo}_issues_{number}_related_branches`
- Tags: None
- Deprecated: `false`
## Parameters
| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| owner | path | true | string | Repository space address (organization or individual's address path) |
| repo | path | true | string | Repository path (path) |
| number | path | true | string | Issue Number (case-sensitive, do not add #) |
| access_token | query | true | string | User authorization code |
## Request Body
No request body.
## Responses
### `200`
Headers:
```json
{}
```
#### `application/json`
Schema:
```json
{
  "items": {
    "description": "Branch name",
    "type": "string"
  },
  "type": "array"
}
```

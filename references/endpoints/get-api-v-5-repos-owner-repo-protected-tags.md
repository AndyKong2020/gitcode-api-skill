# List Project Protection Tags
Source: [https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-protected-tags](https://docs.gitcode.com/en/docs/apis/get-api-v-5-repos-owner-repo-protected-tags)
Category: Tag
- Method: `GET`
- Path: `/api/v5/repos/{owner}/{repo}/protected_tags`
- Operation ID: `get_api_v5_repos_{owner}_{repo}_protected_tags`
- Tags: `Tag`
- Deprecated: `false`
## Parameters
| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| owner | path | true | string | Repository space address (organization or individual's address path) |
| repo | path | true | string | Repository path (path) |
| access_token | query | true | string | User authorization code |
| page | query | false | integer | Current page number: Default is 1 |
| per_page | query | false | integer | Number of items per page: Maximum 100, default 20. |
## Request Body
No request body.
